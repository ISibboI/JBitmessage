Provides the classes and interface to EC domain parameters. They are used by all algorithms of the ec package. EC domain parameters include following items:
EC domain parameters
  • q: the size of the underlying field
    either a prime p or 2m, m integer
  • a: first parameter of the underlying elliptic curve
  • b: second parameter of the underlying elliptic curve
  • G: point of the elliptic curve with order r
  • r: order of point G and primefactor of #E, i.e. r | #E
  • k: cofactor, i.e. k*r = #E
The class CurveParams provides a (transparent) implementation of EC domain parameters. The class ECParameters is used to encode and decode the parameters.
